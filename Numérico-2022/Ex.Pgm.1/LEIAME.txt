O programa foi desenvolvido no Code::Blocks, utilizando o compilador GNU GCC padrão.

Abra a pasta EP e o aplicativo main.

A execução é direta, basta executar main ou abrir e executar o arquivo main.c pelo Code::Blocks que a interface desenvolvida já é apresentada.

Após isso, selecione a opção desejada:

1- Decomposição LU de uma matriz tridiagonal cíclica:

Informe a ordem da matriz e cada diagonal (principal, superior e inferior). O programa então exibe as matrizes L e U.

2- Resolução de um sistema linear com matriz tridiagonal cíclica:

Informe a matriz A e o vetor d e é calculado o vetor x do sistema Ax = d.

3- Demonstração do teste pedido no enunciado:

Selecionando essa opção, o programa calcula e exibe a solução do sistema Ax = d, onde A é a tridiagonal cíclica e d o vetor, tais que ambos seguem as regras de formação fornecidas no enunciado, com ordem n = 20.

Atenção: Nesta opção, aumente a tela do prompt de comando para que a matriz impressa fique de forma correta.
